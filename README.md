# bash-scripts

A collection of bash scripts organized by purpose and learning journey.

## 📁 Directory Structure

This repository contains shell scripts organized into three main categories:

### `/fun`
Fun and entertaining scripts.
- **script.sh** - A placeholder for fun scripts

### `/learning`
Educational bash scripts for learning purposes, covering various bash concepts and scripting techniques.
- **bestdayever.sh** - A cheerful script about having the best day ever
- **eldenring.sh** - Game-themed script
- **for.sh** - Demonstrates bash for loop usage
- **getrichquick.sh** - A humorous script
- **pushups.sh** - Exercise-related script
- **notes.txt** - Learning notes and documentation

### `/utils`
Utility scripts for practical functionality.
- **calculator.sh** - A simple calculator utility

## 🚀 Getting Started

### Prerequisites
- Bash shell (v3.0 or later recommended)
- Linux/Unix environment or WSL on Windows

### Usage

To run any script:

```bash
bash ./path/to/script.sh
